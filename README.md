## Instalar dependencias
```sh
$ make install
```

## Compilar
```sh
$ make
```

## Correr el juego
### Servidor
```sh
$ start server
```
### Cliente
```sh
$ start client
```
## Controles
* Movimiento: Flechas
* Salto: Barra espaciadora
* Toggle música: M
